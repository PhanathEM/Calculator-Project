# 🧮 Responsive Calculator

A modern and responsive calculator built with HTML, CSS, and JavaScript.

## ✨ Features

- Basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)

- Dark Mode / Light Mode
- Theme Persistence using Local Storage
- Responsive Design
- Keyboard Support
- Modern UI
- Mobile Friendly

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## ⌨️ Keyboard Shortcuts

| Key | Action |
|------|--------|
| 0-9 | Numbers |
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| . | Decimal |
| Enter | Calculate |
| Backspace | Delete Last Character |
| Esc | Clear Display |

## 🖥️ Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/calculator.git
```

2. Open the project folder

```bash
cd calculator
```

3. Open `index.html` in your browser

## 🌙 Dark Mode

The calculator automatically saves your theme preference using Local Storage.

When you refresh the page, your selected theme remains active.
